## 1.2 httpd-vhosts.conf新增
```
<VirtualHost *:80>  
    DocumentRoot "D:/www-git"
	<Directory "D:/www-git">
		Options Indexes FollowSymLinks
		AllowOverride None
		Order allow,deny
		Allow from all
	</Directory>
    ServerName localhost
    ##ServerAlias loaclOtherName
</VirtualHost>
```
